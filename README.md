# 🐍 Python Day One Assignment

Hi! 👋
This repository contains three simple Python programs I built to practice basic programming concepts like strings, loops, and functions.

---

## 📌 What’s Inside?

### 🔹 1. Reverse a String

This program takes a word or sentence from the user and reverses it.

**Example:**

```
input: data science
Output:  ecneics atad
```

---

### 🔹 2. Vowel Counter

This program counts how many vowels (a, e, i, o, u) are in a given string.

**Example:**

```
Input: education
Output: 5
```

---

### 🔹 3. CLI Calculator

This is a simple calculator that runs in the terminal.
You can choose an operation and enter numbers to get the result.

**It supports:**

* Addition
* Subtraction
* Multiplication
* Division

**Example:**

```
1.Add  2.Subtract  3.Multiply  4.Divide  5.Exit
Choose: 1
Enter first number: 4
Enter second number: 6
Result: 10
```

---

## ▶️ How to Run the Programs

1. Make sure you have Python installed
2. Clone this repository:

   ```
   git clone <your-repo-link>
   ```
3. Open the folder:

   ```
   cd python-tasks
   ```
4. Run any file:

   ```
   python filename.py
   ```
